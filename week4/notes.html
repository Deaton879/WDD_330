Object-Oriented Programming

One of the cooler things I did not realize JavaScript was able to do, was create objects using a constructor function. Below the difference is demonstrated..


Here is an example code to show the creation of a dice object.


const dice = {

sides: 6,

roll() {

return Math.floor(this.sides * Math.random() + 1)

}

}

And this example shows code which uses a constructor function to create a dice object, instead.


const Dice = function(sides=6){

this.sides = sides;

this.roll = function() {

return Math.floor(this.sides * Math.random() + 1)

}

}
