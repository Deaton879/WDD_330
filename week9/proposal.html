Block 2 Project Proposal: Quote Collector Web App

Purpose

The purpose of this web app is to offer a utility which will allow you to save your favorite quotes, and keep them organized all in one place. The app will also provide the ability to discover new quotes, and even share turn those quotes into post to share on social media.

Procedure

The homepage will show a "quote of the day", which will change each day, and will be accessed using the rest-api: Quotable.
There will also be an option to save the quote of the day to a list of "favorite" quotes.

There will be another page where the user can search for random quotes, search for quotes using search tags (i.e. author, subject, etc.)

The user will be able to manually enter quotes using a form which will collect the following information from the user:

  1. the quote (required)
  2. the author (required)
  3. the source (optional)

The quote will be saved to the user's favorites list.

Finally, the favorites page will allow the user to remove quotes from their list, and will also provide the ability to share any of those quotes on the social media platform of their choosing. This will create a post for the user, and allow them to tailor certain aesthetic aspects of their post before sending (font style, background, etc.)

The user's favorites list will be saved using local storage.
